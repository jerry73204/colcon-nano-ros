# nros-codegen-c

Standalone code generator that emits C bindings for ROS 2 message / service / action types consumed by [nano-ros](https://github.com/NEWSLabNTU/nano-ros)'s `nros-c` and `nros-cpp` APIs. Wraps `rosidl-parser` + a C-emitting Askama template tree.
